# my_photography-
Photo Sharing HTML Page
Overview
This is a simple static web page created using HTML to display and share a collection of photos.
The project was originally created as a basic web development exercise and was also used in my resume portfolio.
Features
*Displays multiple images in a simple layout
*Lightweight static HTML page
*Can be opened directly in a browser


Technologies Used
HTML
Basic CSS (if you used it)
How to Run
Download or clone the repository.
Open the index.html file in any web browser.
Purpose
This project was built to practice basic webpage structure and image embedding using HTML.
